# Prompts

This workspace was generated to build a complete full-stack real-time chat application.

## Backend
- Express server on port 5000
- Socket.io connection handling
- Room-based messaging for General and Tech Support
- Typing events broadcast within a room

## Frontend
- React + Vite client
- Login screen with username and room selection
- Chat view with message list, typing indicator, and compose box
- Socket.io client connection to http://localhost:5000
